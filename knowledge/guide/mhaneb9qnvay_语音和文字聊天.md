# 一、语音和文字聊天功能

语音和文字聊天管理提供贴合玩法的聊天功能，支持根据玩法需要配置多个针对不同玩家或阵营的专属聊天频道。

同一个玩家可以同时在不同的聊天频道中发言

支持利用节点图实时切换玩家聊天频道的权限。

# 二、语音和文字聊天功能管理

通过【系统菜单】——【语音和文字聊天管理】可管理当前关卡聊天频道

全局设置：对语音和文字聊天功能进行全局控制

是否开启实时语音：决定语音聊天功能是否开启

局内是否允许组队语音聊天：决定组队语音聊天功能是否开启

是否开启文字聊天：决定文字聊天功能是否开启

局内是否允许组队文字聊天：决定组队文字聊天功能是否开启

# 三、聊天频道的编辑



添加聊天频道

通过【添加频道】按键，可新增聊天频道

关卡默认带有应用于全部玩家的全局频道，支持修改配置/改名，但无法删除

点击 图标，可重命名，或删除频道



基础信息

|     |     |
| --- | --- |
| 参数 | 说明 |
| _索引_ | 对应聊天频道的唯一标识 |
| _文字聊天初始生效_ | 控制该频道文字聊天功能在关卡初始时的生效情况 |
| _频道应用方式_ | 支持按玩家/按阵营应用<br>当频道按阵营应用时，仅可通过切换玩家阵营调整所属聊天频道 |
| _应用玩家/阵营_ | 控制频道在关卡内的初始覆盖对象，可多选 |
| _显示优先级_ | 控制频道在关卡内聊天系统的显示顺序，数字越大越靠前 |
| _选择图标_ | 控制频道在关卡内聊天系统的图标与颜色 |
| _语音聊天初始生效_ | 控制该频道文字聊天功能在关卡初始时的生效情况 |
| _限制语音范围_ | 控制该频道是否限制语音为范围生效 |
| _收听范围半径(m)_ | 控制该频道的语音收听范围半径，范围1~100m |



快捷消息

点击  可添加快捷文本，点击 可删除文本，至少需要保留1条默认文本

# 五、额外说明

在当前版本中，添加了语音聊天相关功能，同时在界面布局中也增加了局内可以控制语音功能的按钮

考虑到部分奇匠在关卡创作中，可能使用界面控件遮挡住了与语音功能相关的按钮，导致在当前版本中玩家游玩时无法使用语音功能，故在此提醒各位奇匠可以适当调整界面控件的位置，以免影响该功能的正常使用
