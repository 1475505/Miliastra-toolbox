# 一、计时器的功能

用于显示时间进度的界面，包括倒计时、正计时两种表现方式。

# **二、计时器的编辑**

|     |     |
| --- | --- |
| 配置参数 | 说明 |
| _类型_ | 正计时、倒计时 |
| _指定计时器_ | 会枚举符合“类型”的所有预定义全局计时器 |
| _来源实体_ | 默认配置为关卡实体，可选择关卡实体，或者玩家实体 |
