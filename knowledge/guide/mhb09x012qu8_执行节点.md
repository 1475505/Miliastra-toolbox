# 二、战术

## **1.战术：地面对峙**

**节点功能**

造物执行地面对峙行为，四向移动战术，包括静止/前/后/左/右五种方向，可用于来营造与玩家的相持对峙感。

可以通过概率配置创造出只左右绕玩家移动/喜好前进厌恶后退等移动行为，也包含一些与玩家距离调整和避障的配置。

本质上是决策出一个方向，并不会有一个具体的目标点

执行条件：



需要自身或目标在领地范围内



当_是否一定寻路可达_配置为是且不处在空中时，若寻路失败会退出战术

## **3.战术：地面闲逛**

**节点功能**

造物执行地面闲逛行为，在范围内随机行走

执行条件：



受到_造物降频_影响， 距离角色太远的造物不会进入本战术



选到合适的目标点

选点规则：



跟大部分战术不同， _地面闲逛_战术的选点逻辑是在 **可能** 可以执行_地面闲逛_战术但还没进入的时候

因为选到合适的目标点是进入战术的条件之一



真正进入战术以后， 直接执行移动任务， 直到停止移动任务(到达目的地或者撞到障碍物了)后扣除冷却时间， 退出战术



若造物当前位置已经不在_闲逛半径_范围内， _地面闲逛_战术就会选择从当前位置往出生位置的方向， 找距离为_闲逛半径_的点作为目标点



如果支持使用寻路， 会使用寻路探测目标点合法性

| --- | --- | --- | --- |
| **参数类型** | **参数名** | **类型** | **说明** |
| 入参 | 是否执行 | 布尔值 |  |
| 入参 | 速度 | 枚举 | 走、跑 |
| 入参 | 闲逛半径 | 浮点数 | 闲逛距离出生点的最远距离 |
| 入参 | 闲逛最小间隔时长 | 浮点数 | 闲逛冷却时间， 每次退出战术时， 都会在 \[闲逛间隔最小时间， 闲逛间隔最大时间\] 里随机一个时间， 作为冷却时间，冷却时间内不会执行本战术 |
| 入参 | 闲逛最大间隔时长 | 浮点数 |
| 入参 | 单次最小闲逛距离 | 浮点数 | 闲逛距当前点距离 |
| 入参 | 单次最大闲逛距离 | 浮点数 | 闲逛距当前点距离 |

## **4.战术：地面追击**

**节点功能**

造物执行地面追击行为

追击目标战术，会自主选择目标周围合适的位置做目标点

执行条件：

【必须满足】

1.

有目标

2.

自身不在空中

3.

自身或者目标在领地范围内

【任一满足】

4.

当前不在本战术，且自身和目标距离在 \[_触发追击的最小距离_，_触发追击的最大距离_ \] 范围内，才会进入战术

5.

已经处于战术时，自身和目标距离大于_停止追击距离_才会继续执行战术

## **6.战术：脱战后返回出生点**

**节点功能**

造物执行脱战后返回出生点行为

脱离战斗时，回初始点位的战术。战术执行期间，仇恨系统是关闭的， 直到战术结束才会重新打开。

执行条件：



仅支持战前/脱战阶段，并且通常是放在战前的第一个战术，以保证仇恨系统脱战后马上能进入该战术

目标点选择：

目标点选择时机为进入战术那一帧， 不会在战术执行期间修改目标

以下条件从前往后顺序检查， 若满足对应条件， 则按该条内容确定目标点



【执行巡逻】战术执行过，并且配置了_结束时点位设置为出生点_为是时，会以新的出生点作为目标点



选择出生点为目标点

**节点参数**

|     |     |     |     |
| --- | --- | --- | --- |
| **参数类型** | **参数名** | **类型** | **说明** |
| 入参 | 是否执行 | 布尔值 |  |
| 入参 | 移动速度 | 枚举 |  |
| 入参 | 是否传送回出生点 | 布尔值 |  |
| 入参 | 强制触发传送距离 | 浮点数 | 触发前提，需要_是否传送回出生点_配置为是<br>当自身距离出生点位置大于等于配置距离时，会强制触发传送 |
| 入参 | 强制触发传送时间 | 浮点数 | 触发前提，需要_是否传送回出生点_配置为是<br>战术持续时间超过配置时间，仍然未回到出生点，会强制触发传送 |
| 入参 | 屏蔽脱战恢复生命值 | 布尔值 |  |
| 入参 | 战术上下文 | 字符串 |  |
| 入参 | 是否可以打断技能 | 布尔值 | 默认为否 |
