# 一、特效播放组件的功能

特效播放组件包含两部分的功能

1、_节点图_播放的特效必须依赖特效播放组件，没有该组件的情况下无法通过节点图播放特效

2、允许通过组件挂载一些默认的_循环特效_，这些特效会与实体一同创建

特效播放组件上同时可生效多个特效

# 二、特效播放组件的编辑

## **1.添加** 特效播放组件

(1)切换到实体或元件的组件页签

(2)找到或新增一个特效播放组件

特效播放组件是所有单位的默认挂载组件（即实体创建时会默认挂载），因此对于新建的实体或元件，可以在组件页签直接找到特效播放组件

如果不存在，可以通过添加通用组件按钮新增一个特效播放组件

## **2.** 新增特效播放

在特效播放组件的详情页面，可以点击【添加特效】来新增一个特效配置

## **3.** 配置特效播放

|     |     |
| --- | --- |
| **参数名** | **参数说明** |
| _特效播放器序号_ | 特效播放器的序号 |
| _特效播放器名称_ | 特效播放器的名称，同一个实体上的名称不可以重复 |
| _特效资产类型_ | 可选_循环特效_和_限时特效_，决定了使用的特效资产库<br>特效相关资产见[特效](https://act.mihoyo.com/ys/ugc/tutorial//detail/mhe1030vx380) |
| _特效资产_ | 可以选择特效资产。选择了特效资产后，可以在场景中预览播放 |
| _跟随位置_ | 是否跟随组件的拥有者一起运动 |
| _跟随旋转_ | 是否跟随组件的拥有者一起旋转，仅当【跟随位置】为【是】时才生效 |
| _挂接点_ | 特效播放挂接的位置，可以选择实体上已配置的_固有挂接点_或_额外挂接点_ |
| _缩放比例_ | 特效的缩放比例 |
| _偏移_ | 特效播放位置相对于挂接点的偏移 |
