# 一、小地图标识组件的功能

_小地图标识组件_，支持实体通过图标/范围等可视化标识显示在 **预制小地图界面控件** 中

小地图标识组件需要挂载于实体，支持_玩家_、_物件_、_造物_使用

小地图标识可以通过默认配置激活/关闭，也可以通过_节点图_控制指定小地图标识的激活/关闭

# **二、** 小地图标识组件 **的编** 辑

## **1.添加组** 件

(1)在实体或元件编辑界面中，打开组件编辑页签

(2)点击下方的“添加组件”，选择并点击“小地图标识”，成功添加

(3)点击“详细编辑”，展开编辑页

## **2.** 小地图标识 **组** 件的基础信息

|     |     |
| --- | --- |
| 配置参数 | 说明 |
| _初始生效_ | 下拉菜单提供小地图标识枚举<br>通过修改后面的勾选框，可以修改生效情况，修改后生效状态会同步到小地图标识详细编辑页内 |
| _\*小地图标识列表_ | 枚举实体配置的所有小地图标识 |

## 3\. **添加小地图标识**

点击详细编辑可进入小地图标识的编辑



通过点击，添加小地图标识。

添加的小地图标识 **初始生效** 默认开启。



_标记X_，X为 **序号**，作为节点输入项可以调整小地图标识的初始生效参数。

## 4\. 小地图标识的参数说明

### (1)显示设置

|     |     |
| --- | --- |
| 配置参数 | 说明 |
| _初始所有玩家可见_ | 若开启，则小地图标识生效时，满足条件的所有玩家都可以在其小地图界面控件上看到该标识 |
| _跟随物体可见性_ | 若开启，则实体隐藏时，小地图标识一同隐藏 |
| _显示优先级_ | 在同一位置的小地图标识，造成堆叠后，优先级更高的会覆盖优先级低的。数字越大，优先级越高 |

### (2)标记样式

|     |     |
| --- | --- |
| 配置参数 | 说明 |
| _选择类型_ | 分为图标、范围、标记点、玩家标记、造物头像几种，不同类型会带有不同的配置参数 |
| _显示高低差_ | 开启后，高低差超过10m的实体会在小地图内额外标记 |

### (3)选择类型的分类



图标

|     |     |
| --- | --- |
| 配置参数 | 说明 |
| _选择图标_ | 提供多种可供选择的图标 |
| _底图_ | 提供无、圆形 |
| _底图外框颜色_ | 图标底色支持选择，包括指定颜色和逻辑颜色。<br>逻辑颜色包括<br>【敌友关系】敌方红色、友方绿色、自身蓝色<br>【跟随自身阵营】根据自身当前的阵营颜色<br>【跟随所有者阵营】跟随当前实体的所有者阵营颜色 |
| _是否可点击_ | 开启后，展开地图，点击标识点，会显示配置信息 |
| _文本内容_ | 支持配置点击后的右侧显示文本 |



范围

|     |     |
| --- | --- |
| 配置参数 | 说明 |
| _范围样式_ | 支持范围边缘显示为实线、虚线 |
| _颜色_ | 图标底色支持选择，包括指定颜色和逻辑颜色。<br>逻辑颜色包括<br>【敌友关系】敌方红色、友方绿色、自身蓝色<br>【跟随自身阵营】根据自身当前的阵营颜色<br>【跟随所有者阵营】跟随当前实体的所有者阵营颜色 |
| _范围大小(m）_ | 根据配置的尺寸，在小地图界面控件中，按比例显示 |



标记点

|     |     |
| --- | --- |
| 配置参数 | 说明 |
| _样式预览_ | 展示点的样式，不可修改，且点在大地图中不可点击 |
| _颜色_ | 图标底色支持选择，包括指定颜色和逻辑颜色。<br>逻辑颜色包括<br>【敌友关系】敌方红色、友方绿色、自身蓝色<br>【跟随自身阵营】根据自身当前的阵营颜色<br>【跟随所有者阵营】跟随当前实体的所有者阵营颜色 |



玩家标记

|     |     |
| --- | --- |
| 配置参数 | 说明 |
| _样式预览_ | 预览图标为创作者(奇匠)的当前头像，实际生效的图标需要通过节点传入，且玩家标记在大地图中不可点击 |
| _底图外框颜色_ | 图标底色支持选择，包括指定颜色和逻辑颜色。<br>逻辑颜色包括<br>【敌友关系】敌方红色、友方绿色、自身蓝色<br>【跟随自身阵营】根据自身当前的阵营颜色<br>【跟随所有者阵营】跟随当前实体的所有者阵营颜色 |



造物头像

|     |     |
| --- | --- |
| _头像预览_ | 仅造物可以设置此项，仅提供预览，不可编辑 |
| _底图外框颜色_ | 图标底色支持选择，包括指定颜色和逻辑颜色。<br>逻辑颜色包括<br>【敌友关系】敌方红色、友方绿色、自身蓝色<br>【跟随自身阵营】根据自身当前的阵营颜色 |
| _是否可点击_ | 开启后，展开地图，点击标识点，会显示配置信息 |
| _标记名称_ | 标记的名称 |
| _文本内容_ | 支持配置点击后的右侧显示文本 |
