# 一、战斗设置是什么

提供自定义战斗玩法设计更详细的配置，是玩家设置攻击与受击相关信息的手段

# 二、战斗设置编辑

在特化配置页签下，可点击_详细编辑_进入实体的战斗设置

|     |     |
| --- | --- |
| 配置参数 | 说明 |
| _不可被元素附着_ | 配置该物件是否会被元素附着 |
| _不可被目标锁定_ | 配置任何锁定该物件的功能是否会生效 |
| _受击特效_ | 配置物件受到攻击时播放的特效 |
| _被击倒特效_ | 配置物件被销毁或倒下时播放的特效 |

其中受击特效和被击倒特效可以在详细编辑页面进一步配置

|     |     |
| --- | --- |
| 特效配置参数 | 说明 |
| _特效资产类型_ | 仅可选_限时特效_<br>特效相关资产见[特效](https://act.mihoyo.com/ys/ugc/tutorial//detail/mhe1030vx380) |
| _特效资产_ | 可以选择特效资产。选择了特效资产后，可以在场景中预览播放 |
| _缩放比例_ | 特效的缩放比例 |
| _偏移_ | 指定相对位置 |
| _旋转_ | 指定特效的旋转信息 |
