# 基础运动器——让物体以一定规律运动

## 功能示例-1：制作一个持续往返运动的机关装置

### 1.概念简述

**基础运动器：** 通过配置运动规则，让物体以一定规律运动（如：匀速直线、匀速旋转等）

### 2.教学内容

**功能效果：** 做一个可以持续进行往返运动的物体（该功能可不使用节点图，仅面板配置实现），可用作跑酷/解密模式中的场景移动装置；实际应用时可根据需要制作大摆锤、过山车等机关装置

**效果演示：**

### 3.操作步骤

**1.【局内编辑器】添加组件**

创建一个元件作为交互物（教程以移动平台为例），在面板中点击>>>进入组件页，点击>>>选择【基础运动器】即可添加

**2.【局内编辑器】添加触发器**

打开【组件-基础运动器】后，默认不含有运动器，点击即可添加运动器（可同时添加多个运动器制作组合运动的效果）

**3.【局内编辑器】配置面板参数**

配置基础运动器的运动规则，确认后保存元件设置

为达到预期的功能效果教程中配置如下：

|     |     |
| --- | --- |
| 要做的功能 | 对应的配置项 |
| 平台创建后就持续运动（无特殊触发条件） | 【初始生效】开启 |
| 平台沿特定路径往返移动 | 【运动类型】路径运动<br>【路点】配置运动位置和运动时间 |
| 平台持续循环运动 | 【循环类型】调整为循环 |

配置完成后，可在当前界面直接预览当前运动器的效果

此外，还可在面板中配置基础运动器的【运动类型】、【运动速度】等规则，也可以组合多个运动器达到复杂运动的效果，实际使用时可根据需求自由调整

_\*\*配置项说明：_

_运动类型：目前支持【匀速直线】、【匀速旋转】、【朝目标旋转】、【路径运动】4种方式；此外，还可以添加多个不同类型的运动器叠加组合（特殊：【匀速旋转】和【朝目标旋转】不能同时配置）_

**4.【局内编辑器】放置元件，试玩体验**

将制作好的元件放置在场景中，进入试玩即可体验该功能

\*\*更多相关信息可自行查阅【综合指南-基础运动器组件】

## 功能示例-2：踩到特定平台上，开门；离开特定平台，关门

**功能说明：** 玩家踩到特定平台上（进入某区域），打开大门；离开特定平台（离开某区域），关闭大门

**效果演示：**

**实现步骤：**

1.创建一个元件A，作为门

2.在元件A上，添加【组件-基础运动器】，添加2个运动器，分别用作开门和关门；配置运动模式均为【匀速直线】，运动时间、速度可根据需求自由调整（设置参考见下图）

3.可点击【预览当前运动】查看运动效果是否符合预期，确认无误后保存设置

4.在元件A上，添加自定义变量，命名为【是否开门】，类型为【布尔值】，初始值为【否】（实际使用时可根据自身需求自由调整）

_\*\*补充说明：由于仅涉及开/关门两个状态，为避免多人游戏时重复触发开门/关门事件，导致效果出错，因此额外通过自定义变量来管理门的状态_

5.创建一个元件B，作为触发物

6.在元件B上，添加【组件-碰撞触发器】，并配置触发区

7.将元件A放置在场景中，查看并记录其GUID（注意是场景中物件的GUID，而非元件ID）

8.配置元件B的节点图（设置参考见下图），为达到预期的功能效果教程中配置如下：

|     |     |     |     |
| --- | --- | --- | --- |
|  |  | 要做的功能 | 对应的配置项 |
| 功能-1<br>进入区域开门 | 触发场景 | 踩到触发物上时 | 【事件节点】：进入碰撞触发器时（从外部进入，接触到配置的区域时触发） |
| 条件 | 当前门没开 | 【流程控制节点】：判断门当前的状态为“未开门”（自定义变量，是否开门=否） |
| 结果-1 | 开门 | 【执行节点】：使用【激活基础运动器】-激活开门的运动器 |
| 结果-2 | 开门过程中/之后无法再次触发开门 | 【执行节点】：设置门当前的状态为“已开门”（自定义变量，是否开门=是） |
| 功能-2<br>离开区域关门 | 触发场景 | 离开触发物上时 | 【事件节点】：离开碰撞触发器时（从内部出去，接触到配置的区域时触发） |
| 条件 | 当前门打开 | 【流程控制节点】：判断门当前的状态为“已开门”（自定义变量，是否开门=是） |
| 结果-1 | 关门 | 【执行节点】：使用【激活基础运动器】-激活关门的运动器 |
| 结果-2 | 关门过程中/之后无法再次触发关门 | 【执行节点】：设置门当前的状态为“未开门”（自定义变量，是否开门=否） |

此外，还可以使用其他基础运动器节点制作功能（如：开门后触发xx效果、施放时停技能使物体暂停运动等）

9.将节点图挂载在元件B上，保存设置

10.将元件B放置在场景中

11.进入试玩，即可体验该功能
