# 特效播放器组——让物体呈现不同效果

## 功能示例-1：制作一个持续燃烧的火炬

### 1.概念简述

**特效：** 制作各类视觉表现的美术资产（编辑器和千星沙箱内置了一批预设特效）

**特效播放器组：** 通过添加特效和特效规则，让物体呈现不同的效果（如：发光的门、爆炸效果、子弹轨迹等）

### 2.教学内容

**功能效果：** 做一个持续燃烧的火炬（该功能可不使用节点图，仅面板配置实现），可用作跑酷/解密模式中的场景装饰物或解密道具

**效果演示：**

_\*\*物件-火炬本身是没有燃烧特效的，添加特效后可呈现持续燃烧的效果_

### 3.操作步骤

**1.【局内编辑器】添加特效**

创建一个元件作为交互物（教程以火炬为例），在面板中点击>>>进入组件页，打开【特效播放器组】>>>点击【添加特效】

_\*\*所有可配置特效的单位创建时都默认含有【组件-特效播放器组】，无需额外添加；如果自己删除了，可以通过添加组件再次添加_

**2.【局内编辑器】配置面板参数**

为达到预期的功能效果教程中配置如下：

|     |     |
| --- | --- |
| 要做的功能 | 对应的配置项 |
| 特效持续循环 | 【特效资产类型】选择循环特效 |
| 火焰特效 | 【特效资产】选择合适的火焰类效果 |
| 火焰的位置在火炬尖端 | 【偏移】调整至合适位置 |

配置完成后，可在当前界面直接预览特效播放效果

此外，还可在面板中配置特效【特效类型】、【跟随位置/旋转】、【挂接点】、【缩放比例】等规则，实际使用时可根据需求自由调整

_\*\*配置项说明：_

_特效类型：分为循环特效和限时特效两种，并且每种类型对应的特效资产也不同_

_循环特效：持续播放；_

_限时特效：播放一次后消失_

**3.【局内编辑器】放置元件，试玩体验**

将制作好的元件放置在场景中，进入试玩即可体验该功能

_\*\*更多相关信息可自行查阅【综合指南-特效播放器组】_

## 功能示例-2：制作一个交互点火的开关

**功能说明：** 交互物件A，给物件B添加特效（交互机关使火炬点燃）

**效果演示：**

_\*\*演示中的文本显示，需达到Lv.3后解锁【自定义文本外显】功能，教程中以效果展示为主_

**实现步骤：**

1.创建一个元件A，作为火炬

2.在元件A的【组件-特效播放器】中选择要添加的特效和位置等参数信息（此处非配置，而是通过预览确认添加后的效果，实际面板配置为空）

_\*\*教程中延用功能示例-1的特效与参数配置为例，实际制作时可根据需要灵活调整_

--循环特效

--特效id:189

--偏移（0,1.2,0）

3.将元件A放置在场景中，确认其GUID

4.创建一个元件B（教程中选择拉杆机关为例），作为交互物

5.在元件B上，添加【组件-选项卡】，配置选项和触发选项区域范围

6.配置元件B的节点图（设置参考见下图），为达到预期的功能效果教程中配置如下：

|     |     |     |     |
| --- | --- | --- | --- |
|  |  | 要做的功能 | 对应的配置项 |
| 功能<br>交互后点火 | 触发场景 | 交互选项后触发 | 【事件节点】：选项卡选中时（交互选择“任意”选项时触发） |
| 条件 | / | / |
| 结果-1 | 火炬点火 | 【执行节点】：在元件A上挂载循环特效，特效资产与位置等即为步骤2和3中确认的信息 |
| 结果-2 | 拉杆机关变化表示交互成功 | 【执行节点】：使用【设置预设状态】制作拉杆效果 |
| 结果-3 | 使用机关后不能再次使用 | 【执行节点】：使用【激活/关闭选项卡】将选项关闭达到无法再次交互的效果 |

7.将配置的节点图挂载在该元件上，保存元件设置

8.将元件B放置在场景中，进入试玩即可体验该功能
