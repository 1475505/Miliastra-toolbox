# 主镜头——更改游戏视角

## 功能示例-1：制作不同视角下的游戏镜头

### 1.概念简述

**主镜头编辑：** 是游戏中用于呈现玩家视角的工具，可通过定义镜头模板来定义镜头的规则（如：通过更换镜头模式，切换为第一人称/第三人称/俯视角等）

### 2.教学内容

**功能效果：** 制作不同的镜头，用于不同视角模式的游戏（教程中以俯视角为例进行演示）

**效果演示：**

### 3.操作步骤

**【局内编辑器】更改镜头参数**

点击局内编辑器左上角（或直接按ESC键）>>>呼出左侧面板-ESC菜单栏，点击>>>进入主镜头编辑界面，即可更改镜头属性（可通过官方预设的镜头模式快捷调整）

为达到预期的功能效果，教程中配置如下：

|     |     |
| --- | --- |
| 要做的功能 | 对应的配置项 |
| 制作俯视角 | 【基础设置-模式】使用2.5D镜头 |
| 对职业x生效 | 【默认生效目标】调整为所需更改的职业 |

此外，还可在【职业】中配置镜头（两处配置效果一致，任选一处配置即可）

_\*\*更多相关信息可自行查阅【综合指南-镜头】_

## 功能示例-2：交互选项卡后，切换镜头模式

**功能说明：** 制作一个镜头切换装置，当玩家交互选项后，可根据交互选项切换为第一人称或俯视角

**效果演示：**

_\*\*演示中的文本显示，需达到特定奇匠等级后解锁【自定义文本外显】功能，教程中以效果展示为主_

**实现步骤：**

1.配置2个主镜头，分别对应模式为【第一人称镜头】和【2.5D镜头】（可对镜头名称进行自定义，用于后续节点图配置使用）

2.创建一个元件，作为交互物

3.添加【组件-选项卡】，并配置选项和触发区

4.节点图配置（设置参考见下图）

_\*\*图中的【镜头模版名称】即为步骤1中进行自定义后的镜头名称_

5.将节点图挂载在元件上，保存设置

6.将元件放置在场景中

7.进入试玩，即可体验该功能
