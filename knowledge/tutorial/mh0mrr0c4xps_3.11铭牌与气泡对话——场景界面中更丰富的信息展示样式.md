# 铭牌——游玩时，场景界面中的可视化信息

## 功能示例-1：制作常驻显示的指引提示

### 1.概念简述

**铭牌：** 通过配置类型、内容与导航指引，可制作例如：生命值、等级等可视化信息展示，也可以提供方向指引、距离显示等提示引导表现。

### 2.教学内容

**功能效果：** 制作一个常驻显示的任务指引，当玩家靠近一定范围后，a有导航提示距离与方向，接近后提示任务信息

**效果演示：**

_\*\*演示中的文本显示，需达到特定奇匠等级后解锁【自定义文本外显】功能，教程中以效果展示为主_

### 3.操作步骤

**1.【局内编辑器】添加组件**

①创建一个元件作为任务NPC（教程以积木玩偶为例），在面板中点击>>>进入组件页，点击>>>选择【铭牌】即可添加

②此外，为了便于快速调整铭牌的生效位置（铭牌的生效位置可配置于自定义挂接点），添加“自定义挂接点”组件

**2.【局内编辑器】添加自定义挂接点**

打开【组件-自定义挂接点】后，点击下方添加自定义挂接点即可添加（按需设置其“名称”、“偏移”、“缩放”）

**3.【局内编辑器】添加铭牌**

打开【组件-铭牌】后，点击即可添加，可以修改其“挂点”、“可见半径”、“类型”、“导航指示器”等设置

**4.【局内编辑器】配置面板参数**

添加【铭牌】后可对铭牌与导航指示器进行调整，为达到预期的功能效果教程中配置如下：

|     |     |
| --- | --- |
| 要做的功能 | 对应的配置项 |
| 铭牌在头顶显示 | 【挂点】调整为“头顶”<br>\*\*头顶为此前预设的自定义挂点的名称，根据实际配置选择即可 |
| 靠近一定范围可以看到铭牌信息提示 | 【可见半径】调整为“5”<br>\*\*根据实际需要设定即可 |
| 铭牌有文字与图标提示 | 添加2个铭牌内容；<br>【类型】分别设置为“文本框”和“图标”<br>\*\*根据实际需要设定文字与图标内容与位置偏移即可 |
| 在一定范围内给玩家导航引导（用于提示方向） | 启用导航指示器并进行设置【最大可见半径】与【最小可见半径】 |
| 导航指示器有文字（显示与玩家距离）与图标提示 | 添加2个指示器内容；<br>【类型】分别设置为“文本框”和“图标”<br>显示距离通过【插入变量-数据-与玩家距离】实现<br>\*\*根据实际需要设定文字与图标内容与位置偏移即可 |

**5.【局内编辑器】放置元件，试玩体验**

将制作好的元件放置在场景中，进入试玩即可体验该功能

_\*\*更多相关信息可自行查阅【综合指南-组件-铭牌】_

# 气泡对话——游玩时，场景界面中的对话信息

## 功能示例-2：制作一段循环播放的引导对话

### 1.概念简述

**气泡对话：** 通过配置气泡文本，可便捷制作可视化的对话信息表现。

### 2.教学内容

**功能效果：** 制作一段循环播放的对话信息，当玩家靠近一定范围后可以看见这段对话

**效果演示：**

_\*\*演示中的文本显示，需达到特定奇匠等级后解锁【自定义文本外显】功能，教程中以效果展示为主_

### 3.操作步骤

**1.【局内编辑器】添加组件**

①创建一个元件作为任务NPC（教程以积木玩偶为例），在面板中点击>>>进入组件页，点击>>>选择【文本气泡】即可添加

②此外，为了便于快速调整铭牌的生效位置（铭牌的生效位置可配置于自定义挂接点），添加“自定义挂接点”组件

**2.【局内编辑器】添加自定义挂接点**

打开【组件-自定义挂接点】后，点击下方添加自定义挂接点即可添加（按需设置其“名称”、“偏移”、“缩放”）

**3.【局内编辑器】添加气泡文本**

打开【组件-气泡】后，通过【添加气泡文本】可添加新的对话内容，用于制作完整的长段对话（此处教程中以制作多段对话为例进行演示）。

此外，点击可添加新的气泡（用于切换全新的对话内容），教程中不额外演示。

_\*\*气泡对话组件同一时间仅生效1个气泡对话_

**4.【局内编辑器】配置面板参数**

添加【铭牌】后可对铭牌与导航指示器进行调整，为达到预期的功能效果教程中配置如下：

|     |     |
| --- | --- |
| 要做的功能 | 对应的配置项 |
| 气泡在头顶显示 | 【挂点】调整为“头顶”<br>根据需要调整偏移位置<br>_\*\*头顶为此前预设的自定义挂点的名称，根据实际配置选择即可_ |
| 靠近一定范围可以看到气泡对话 | 【可见半径】调整为“5”<br>_\*\*根据实际需要设定即可_ |
| 气泡有文字引导内容 | 添加2个气泡文本；<br>设置“文本框”内容<br>_\*\*根据实际需要设定文字内容即可_ |
| 循环播放该段对话 | 【是否循环播放】开启 |

**5.【局内编辑器】放置元件，试玩体验**

将制作好的元件放置在场景中，进入试玩即可体验该功能

_\*\*更多相关信息可自行查阅【综合指南-组件-气泡对话组件】_

# 拓展应用

该模块仅作为教学课程所能拓展应用的功能简述，涉及到多模块联动，以操作步骤演示为主，了解进阶用法即可

## 功能示例-3：任务引导标识和气泡对话的组合应用

**功能说明：** 玩家靠近一定距离后提示任务标识，靠近后提示可与NPC对话，交互后NPC播放对话内容

**效果演示：**

_\*\*演示中的文本显示，需达到特定奇匠等级后解锁【自定义文本外显】功能，教程中以效果展示为主_

**实现步骤：**

1.制作1个元件作为任务NPC（教程以积木玩偶为例），添加【组件-选项卡】，配置选项“打招呼”，配置合适的选项触发区域

2.添加【组件-自定义挂接点】，添加一个自定义挂接点，根据需要配置其位置偏移

3.添加【组件-铭牌】，配置铭牌的图标、文本与导航指示器（与本教程功能示例1配置一致，此处不额外演示）

4.添加【组件-气泡】，配置气泡的文本内容（与本教程功能示例2配置一致，此处不额外演示）

需要注意：由于该功能的气泡是通过主动交互显示，因此需关闭初始生效的气泡

5.NPC的节点图配置“交互后播放气泡对话”效果（设置参考见下图），为达到预期的功能效果教程中配置如下：

\*\*该节点图挂载在【NPC元件】上

|     |     |     |     |
| --- | --- | --- | --- |
|  |  | 要做的功能 | 对应的配置项 |
| 功能<br>交互选项后，播放气泡对话 | 触发场景 | 进行交互操作后 | 【事件节点】：选项卡选中时（交互选择“任意”选项时触发） |
| 条件 | / | / |
| 结果 | 播放气泡对话 | 【执行节点】：让NPC播放气泡对话1 |

6.将节点图分别挂载在元件上，保存设置

7.将制作完成的元件摆放在场景中，进入试玩即可体验该功能
