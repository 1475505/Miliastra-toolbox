# Q：为什么编写节点图后点击试玩，编写的逻辑不生效，并且结束试玩后发现节点图回退到未编辑前的状态？

A：这是由于您在关卡还处于试玩状态时进行了节点图编写，试玩过程中编写的节点图不会被保存。建议结束试玩后再继续进行节点图的编写，以免丢失重要逻辑。

# Q：房主默认是玩家1吗？

A：现阶段游戏中的玩家1-8是根据进入关卡的顺序来设置的，因此就算是房主开启游戏，如果有玩家比房主先Loading完成，先进入的玩家就会依次为玩家1-8。

# Q：我可以在游戏中设置假人玩家来测试多人效果吗？

A：现阶段暂不支持，如果需要测试多人效果，需要找其他真人玩家来参与多人测试。

# Q：我在哪里查看【打印字符串】节点打印出来的内容？

A：在千星沙箱【窗口】中打开【日志】，实际运行中的字符串会显示在日志信息中。

# Q：为什么我在【日志】中已经勾选了我想调试的节点图，但是实际运行依旧不显示日志信息？

A：可能是因为在勾选后，您没有选择【确认选择】导致的。

​

# Q：为什么我还没有进入游戏，但是其他先进入的玩家已经可以开始玩了？

A：有玩家进入之后，游戏就会默认开始，因此最好单独写一个确认所有人已进入并准备的功能。

# Q：为什么关卡游玩过程中出现了反复传送或者反复打开卡牌选择器？

A：部分异常可能会导致旅行者在游玩时，遇到反复进行传送，或者反复打开卡牌选择器的问题。遇到该问题的旅行者请退出游戏，并结束游戏进程，等待十五分钟后重新进入游戏即可恢复。
