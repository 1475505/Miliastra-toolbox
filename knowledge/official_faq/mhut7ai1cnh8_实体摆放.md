# Q：实体摆放有没有上限？具体上限是多少？

动态单位摆放上限为1000，静态单位无上限。（静态单位指无法绑定组件和节点图功能的纯实体。）![](https://act-webstatic.mihoyo.com/ugc-tutorial/faq/cn/zh-cn/mhut7ai1cnh8/15b02cf5-68c0-4bbe-8056-9dcb066d4826.png)

# Q：在哪里可以查看场景中已经摆放的实体列表？

现阶段可以通过千星沙箱下方的【场景编辑空间实体】列表查看已经放置的实体，通过双击可以直接在局内编辑器中跳转到实体所在位置。

# Q：我能不能对多个实体进行编组？

可以通过设置实体布设组功能对多个实体进行编组。

# Q：选择实体时，操作杆只进行缩放操作，无法移动或者旋转是为什么？

在“局内编辑器”选择“投射物”后，再选择物件、触发盒等，会出现“操作杆只能进行缩放操作的问题”；如奇匠遇到该问题，请通过空格键手动切换到移动/旋转/三合一操作杆。

# Q：我想要通过操纵杆对实体进行等比缩放操作，应该怎么操作？

A：首先，点击目标实体，通过空格键将操纵杆切换为缩放状态：

随后，点击操纵杆的中心圆点，长按鼠标左键即可进行等比缩放。

# Q：造物（怪物）能不能改变大小？

造物（怪物）大小为固定值，不支持修改造物（怪物）大小。

# Q：为什么造物（怪物）的脱战存在异常？

“禁止移动”单位效果作用于敌人时，可能会导致敌人脱离战斗后表现异常，移除该状态后即可恢复正常。
