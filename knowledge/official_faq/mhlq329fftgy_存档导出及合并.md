# Q：为什么我点击导出存档后，在弹出的保存路径页面点击【保存】不生效？

您没有对导出的文件进行命名导致的，需要编辑好文件名后，保存按钮才能被按下。

![](https://act-webstatic.mihoyo.com/ugc-tutorial/faq/cn/zh-cn/mhlq329fftgy/b77f854c-8e61-436f-881e-a9999a4e9a42.png)

# Q：为什么我导入外部资源后，没有在现有元件库/节点图中找到导入的外部资源？

导入外部资源后，外部资源统一放置在资产库中，根据需要手动单独转存。

# Q：为什么日志中不打印【实体创建时】事件触发的日志？

在游戏开始时就创建的实体中的【实体创建时】会无法在日志中显示，这是由于实体创建时日志功能还没有接入，但是实际逻辑是正常运行的。

# Q：为什么我导出导入的资源有部分逻辑丢失了？

在“局内编辑器”使用“本地资产加载”或“资源导出”时，若相关资源之间存在关联内容，需要将关联的内容同时导入或导出，否则会丢失部分资源资产之间的引用信息；

# Q：上传存档或者文件的过程中，编辑器会未响应?

当前在关卡试玩和关卡上传时，通过来回切换窗口离开/返回编辑器，有概率导致出现未响应状态的情况，耐心等待恢复即可。
